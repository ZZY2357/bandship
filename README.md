# bandship

#### 介绍
> 军舰游戏
>
> 由Python编写

#### 软件架构

```
bandship
	src
		__init__.py			程序主入口
		game.py				游戏主流程控制
		make_grid.py		操作二维数组及游戏地图
		render.py			渲染二维数组
		think_user.py		操作用户输入
	LICENSE		开源协议
	README.md	你现在所看到的
```
